# Priparing items
* must
    * RX72N Envision Kit x 1
    * USB Cable(USB Micro-B --- USB Type A) x 1
    * micro SD Card x 1
    * Windows PC x 1
* option
    * none

# Procedure of firmware update
1. Download updata
    * Download latest updata (extension = rsu)
        * [link](https://github.com/renesas/rx72n-envision-kit/wiki#table-of-firmware-for-update)
1. Store the updata (extension = rsu) to SD card
1. Insert SD card to RX72N Envision Kit SD card slot
1. Start Firmware update via SD card demo
    * Please refer to Firmware Update via SD Card demo on following URL
        * [link](https://github.com/renesas/rx72n-envision-kit/wiki/Confirm-factory-image-behavior#benchmark-demo-bank-b)
1. Touch "userprog.rsu" button on lefg on display
1. Touch "Update" button on right-low on display
1. Reset will be executed in automatically after 100% progressing
1. Start updated firmware
